What is CI/CD?

CI/CD is a way to make software work faster and better. CI means Continuous Integration, where code is checked automatically after we upload it. CD means Continuous Deployment, where the website is published automatically. This helps reduce mistakes and saves time. It also makes updates appear quickly for users.

My Pipeline

When I push my code to the main branch, GitHub Actions starts automatically. It checks that the required files exist in the project. After that, GitHub uploads the website files and deploys them to GitHub Pages. Then my live website gets updated online.
